# Зерно 3D models

The GLB files in this directory come from the **Kenney Food Kit**:

- Source: https://kenney.nl/assets/food-kit
- License: CC0 1.0 Universal (public domain dedication)
- License text: https://creativecommons.org/publicdomain/zero/1.0/

Files used by the landing page:

- `coffee-cup.glb` — Kenney `cup-coffee.glb`, reused for latte, filter coffee, and the hero.
- `cocoa-mug.glb` — Kenney `mug.glb`.
- `croissant.glb` — Kenney `croissant.glb`.
- `cheesecake.glb` — Kenney `cake.glb`.
- `cinnamon-roll.glb` — Kenney `donut.glb`, used as a close low-poly pastry match.
- `Textures/colormap.png` — shared Kenney Food Kit color map referenced by the GLB files.
